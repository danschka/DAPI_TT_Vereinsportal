# TT-Vereinsportal

Moderne Webplattform für einen Tischtennisverein auf Basis von Blazor und .NET.

## Features

### Öffentliche Bereiche
- Bildergalerie
- Sponsorenanzeige
- Downloadbereich
- Mitgliedsantrag
- Responsive Webanwendung

### Adminbereich
- Bilder hochladen/löschen
- Sponsoren verwalten
- Dokumente hochladen
- Mitgliedsanträge verwalten
- Passwortgeschützter Adminbereich

## Technologien

- Blazor Web App
- ASP.NET Core
- Entity Framework Core
- SQLite
- MailKit
- C#
- Razor Components
